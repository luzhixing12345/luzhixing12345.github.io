前言

CSAPP 是计算机专业的一门非常重要的专业课,绝大部分学校将这门课程开在大二. 奈何笔者水平有限,彼时只觉云山雾绕而不得其法,整个的学习过程颇有囫囵吞枣之感. CMU教授精心准备了11个相关实验,因学时与课程安排等原因只安排做了其中之4(data bomb buffer shell).恰得此空闲,打算完整的学习一下CSAPP的相关内容,并且完成所有的11个课程实验,遂有此系列.

笔者会简单的过一下基础知识,包括linux环境搭建,linux常用命令介绍,vim介绍等等,如果读者已经比较熟悉了可以直接跳至对应实验部分

熟悉命令行操作

Windows系统想必已经比较熟悉了,界面窗口(windows)做的很优秀,我们可以通过鼠标完成大部分的交互操作

但是Linux系统比较特殊,它主要依赖于命令行输入命令执行相关操作,我们可以使用一串串指令完成几乎所有任务.这时候我们可以抛弃鼠标,只使用键盘.

下面介绍一些常用的基本命令

  1. clear 清空所有的终端输出

    ctrl + L

  2. ls 列出目录的内容及其内容属性信息

    ls -a 列出所有文件,包含一些隐藏文件

  3. cd 切换目录

    • cd 回到根目录
    • cd … 回到上一级目录
    • cd {目录名} 进入当前路径下的文件夹

  4. mv 移动文件

    mv ../a.txt b.txt

    这条命令是指将上一级目录的 a.txt 文件移动到当前目录并且改名为 b.txt

  5. cp 复制文件

    用法同mv但是cp保留原文件mv将原文件移动走了

  6. tar 解压文件

    相关链接

Vim编辑器入门

  1. 使用vim打开一个文件(文件名为test.txt)

    vim test.txt

    如果显示vim不存在则需要先安装vim:

    sudo apt install vim

  2. 进入vim之后可以使用 ctrl + f 快速下翻页, ctrl + b 上翻页

  3. 按下键盘的 i 进入插入模式(左下角会出现-- INSERT --的字样),表示当前状态是可以编辑的,在这个状态下修改你的文件

  4. 修改完成后,按下 esc 左下角-- INSERT --消失,接着输入 :wq

    w表示保存 q表示退出

    输入之后按下回车文件修改完毕保存成功,回到之前的界面
