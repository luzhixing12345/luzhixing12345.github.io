安装中文输入法

https://blog.csdn.net/mr_sudo/article/details/124874239

1. 添加中文语言支持

Settings --> Region & Language --> Manage Installed Languages

勾选Chinese(Simplified), install

2. 安装fcitx输入法框架

sudo apt update
sudo apt install fcitx

3. 设置fcitx为系统输入法

这里笔者的是fcitx4

Keyboard input method system选择fcitx4，然后点击Apply System-Wide

4. 设置fcitx开机启动，并卸载ibus输入法框架

sudo cp /usr/share/applications/fcitx.desktop /etc/xdg/autostart/
sudo apt remove --purge ibus

5. 安装搜狗输入法

到搜狗linux下载安装包

选择版本时请注意,目前绝大部分电脑平台应该都是x86架构的,你可以通过如下命令查看平台架构

uname -m

安装

sudo dpkg -i <package name>

这里的package name进入下载目录使用tab补全即可

安装搜狗输入法依赖

sudo apt install libqt5qml5 libqt5quick5 libqt5quickwidgets5 qml-module-qtquick2
sudo apt install libgsettings-qt1

6. 重启电脑，调出搜狗输入法

重启电脑之后就可以在右上角看到键盘的图标了

点击input method,这时候应该是没有搜狗输入法的

点击configure,左下角加号,取消勾选Only Show Current Language,在下面的宽中输入sogou或者滚动条拉到最下面,添加即可

之后就可以在input method中找到sougou了,用力准确的点击一下,图标会切换为搜狗图标,然后就可以输入中文了

Vscode

不要使用ubuntu的software来安装vscode,存在中文无法输入的问题,到官网下载linux版本的

进入官网,对于ubuntu下载deb即可

Chrome

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb

接着就可以找到chrome了

不过笔者虽然成功安装但是并没有成功打开,在vscode中ctrl点击一个链接之后莫名其妙就成功打开了…
