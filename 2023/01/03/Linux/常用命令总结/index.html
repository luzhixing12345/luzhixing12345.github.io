常用命令总结

本文最后更新于：8 天前

  • 查询linux命令
  • 解释shell

查看操作系统版本

cat /etc/os-release

find: 搜索

在文件夹中递归的搜索某个文件名

find <PATH> -name "NAME"

# 从此目录下递归的搜索文件名为cdev.h的文件
find ./ -name "cdev.h"

查找一个带文件夹的文件名

例如要找 config/have_deprecated_namespace.hh, 即已知在 config/ 下

find <PATH> -name <NAME> -path "*/config/*"

# find /home/kamilu/gem5 -name have_deprecated_namespace.hh -path "*/config/*"

查找一个库的文件的位置

例如要找 gtest/gtest.h 文件的位置, 这是一个C++库googletest的库函数的文件

  1. 使用locate

    sudo apt plocate
    locate gtest/gtest.h

  2. 全局搜索(很慢)

    sudo find / -name <NAME> 2>/dev/null | grep '<NAME>$'

# sudo find / -name gtest.h 2>/dev/null | grep '/gtest.h$'

    这个命令将在整个文件系统中递归搜索 gtest.h 文件，并输出包含 /gtest.h 的文件路径。 2>/dev/null 表示将标准错误输出重定向到空设备，以避免在没有权限的目录中出现大量的错误信息。 grep ‘/gtest.h$’ 表示仅输出以 /gtest.h 结尾的文件路径，以过滤掉不相关的文件。

    请注意，这个命令可能需要一段时间来完成，具体取决于您的文件系统大小和搜索范围。如果您知道该文件应该在哪个目录中，您可以将 find 命令的搜索范围限制为该目录，以提高搜索效率。

查找指定类型的

find <PATH> -type f

在 PATH 文件夹中查找所有普通文件（不包括目录和符号链接）。

  • -type f 查找所有文件
  • -type d 查找所有目录
  • -type l 查找所有符号链接

查找修改时间在指定时间范围内的文件

find <PATH> -mtime -7

在 PATH 文件夹中查找最近 7 天内修改过的文件。

  • -mtime +7 查找 7 天前修改过的文件
  • -mtime 7 查找恰好在 7 天前修改过的文件

查找文件的内容

查找当前目录下所有.h文件中包含 struct aaa { 内容的文件

find ./ -name "*.h" | xargs grep "struct aaa {"

在当前文件夹中查找包含 “search term” 的文件内容。此命令使用 grep 命令查找文件内容，并使用 -exec 选项执行该命令。

find ./ -type f -exec grep "search term" {} \;

查找符号链接指向的文件

find <PATH> -type l -ls

在 PATH 文件夹中查找所有符号链接，并显示它们指向的文件的详细信息。此命令使用 -ls 选项显示符号链接指向的文件的详细信息

查找文件大小在指定范围内的文件

find <PATH> -type f -size +10M -size -100M

在PATH文件夹中查找文件大小在 10 MB 和 100 MB 之间的文件。此命令使用 -size 选项查找文件大小在指定范围内的文件，

  • 使用 + 表示大于
    • 表示小于
  • M 表示以 MB 为单位。

du: 查看大小

du -sh <PATH>

# du -sh /home/kamilu/gem5
  • -s 选项表示仅显示汇总信息，而不显示每个子目录的详细信息。
  • -h 选项表示以易于阅读的格式显示文件大小，例如 GB、MB、KB 等。

scp: 传输文件

将主机处文件传到远程服务器

scp ./test.txt root@kamilu.top:/root

将远程服务器的文件传到主机

# 主机的终端中执行
scp root@kamilu.top:/root/text.txt /home/kamilu
  • scp -r 传输文件夹

tar: 压缩解压

解压

tar xf <TAR.gz>

压缩文件夹

tar -zcf a.gz /home/kamilu/a
