# 个人博客

该仓库是基于Github使用hexo搭建的个人博客,用于记录和分享计算机知识,所有内容在master分支中

个人博客访问: [luzhixing12345.github.io](https://luzhixing12345.github.io/)

## 一些思考

写博客的目的是什么?

- 希望可以通过笔记记录所学知识,加深印象
- 希望可以对你有所帮助
